# SupplyTrack
The project analyzes shipment times, order volumes, product demand, and operational bottlenecks to support data-driven decision-making.

# Modelo conceptual – SupplyTracker

Entidades principales

Clientes
Productos
Órdenes
Detalle de órdenes
Envíos
Transportistas
Estados de envío
Ubicaciones

# Relaciones clave:

Un cliente hace muchas órdenes
Una orden tiene muchos productos
Una orden puede tener uno o varios envíos

Un envío tiene un transportista y un estado
